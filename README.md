# Booking Flight app
A place where the users will find the list of available flights and booked a flight

## Tech Stack

- ReactJS
- Hooks and CustomHooks
- React-router
- Dynamic Routing
- React-Router-Dom
- Firebase to storage bookings
- Sweetalert2 
- Date-fns format dates
- Axios
- CSS styles
